




#
进程就是运行中的程序  

线程是进程中的一个执行单元，是程序执行的最小单位

## 单线程
单线程 ≈ 串行（在一条线程上）  
在一条线程中按顺序一条接一条执行

## 多线程
多线程指的是多条执行线同时存在：  

并发（concurrent）：多个线程都在推进，时间上可以重叠。在单核上往往是操作系统分时切换，微观上仍是轮流执行，但宏观上多个任务在“同时进行”。  
并行（parallel）：在多核上，不同线程可以真的在同一时刻在不同核心上跑，这是并行。

# thread 的构造形式
```cpp
std::thread t1(可调用对象, 参数1, 参数2, ...);
```

# 示例
```cpp
#include <thread>
#include <iostream>
using namespace std;
void func1(){
    cout<<"thread";
}
void func2(int a, int b){
    cout<<a+b;
}
int main() {
    //先创建线程对象,使用join()等待线程结束
    thread t1(func1);
    t1.join();
    //给线程函数传参
    thread t2(func2, 1, 2);
    t2.join();
    return 0;
}
```

# 补充的注意事项
1.join() 的位置决定并发关系
写在两个 thread 之间，就等于“先做完前一个再开下一个”
```cpp
//这种两个线程是并发的
thread t1(func1);
thread t2(func2, 1, 2);
t1.join();
t2.join();
```

```cpp
//这种两个线程是串行的
thread t1(func1);
t1.join();
thread t2(func2, 1, 2);
t2.join();
```

2.detach()
```cpp
//将t1线程对象与主线程分离,t1在后台运行,不会影响主线程
t1.detach();

```

3.joinable()
```cpp
bool is_joinable=t1.joinable();//返回一个bool值，是否可以使用join()或detach()函数
```
