
# atomic
atomic用于多线程环境下的原子操作，提供一种安全方式来访问和修改共享数据。避免数据竞争  
对比mutex，atomic不需要锁机制，性能更高，但只能用于简单的数据类型和操作。 

```cpp
#include <thread>
#include <iostream>
#include <atomic>
using namespace std;
atomic<int> a(0); 
void test(){
    for(int i = 0; i < 1000000; i++){
        a++;
    }
}
int main() {
    thread t(test);
    thread t1(test);
    t1.join();
    t.join();
    cout << a << endl;
    return 0;
}
